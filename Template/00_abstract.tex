\chapter*{要約}

要約本文は，00\_abstract.texに記入する．以下は，注意事項であり，各自の論文では，削除すること．

次の内容を,500字程度で記載する.
\begin{itemize}
\item 研究の「目的」
\item 研究の「具体的な方法」：実験方法，解析方法について，具体的に記述する．
\item 研究の「具体的な結果」：実験結果，解析結果を具体的に記述する．数値による記述．
\item 研究の「適用範囲」：実験，および解析結果が保証される範囲について具体的に述べる．
\end{itemize}

書き方の例：

（目的）野外における衝撃波の可視化計測を評価するため，（方法）Background-Oriented Schlieren(BOS)法を室内と屋外において定量的可視化計測を行った．この手法はカメラ，光源と可視化対象の屈折率変化を際立たせるための背景画のみの簡易的な光学系で実施でき，屈折率変化による背景画の歪みを画像処理で算出する．本報での背景画は，屋内実験では黒と白のランダムドット模様の人工背景画，屋外実験では木々，草地や地表面などの自然背景画を使用した．実験は高エネルギー物質で衝撃波を生成させ，高速度ビデオカメラで撮影を行った．画像処理は相互相関法を用いて，ガウス分布による補完で精度を向上させるサブピクセル解析も導入した．（結果）その結果，人工背景画では2次衝撃波や負圧領域を確認できる精度を得た．また，自然背景画とした野外試験においても2次衝撃波を確認できた．（適用範囲）ただし，木々や草地の階調値が単調であるために人工背景画ほどの精度を得るに至っていない．また，背景画の階調値変化の大きい木々や草地では可視化撮影ができるものの，階調値変化の小さい地表面や空では良好な可視化画像が得られないことも明らかにした．（成果の反映先）野外における大規模衝撃波現象計測がBOS法によって定量的に可能であることを示したこの結果は，火薬類の安全性研究，高速飛しょうする航空宇宙機の実機試験に対し，適用可能であると考える．（608文字）
