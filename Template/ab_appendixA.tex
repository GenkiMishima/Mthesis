\chapter{タイトル記入}
「付録」には，以下のような内容の記述を本文と切り離して記述する．

\begin{itemize}
\item 本文の基礎理論の中で登場した関係式の導出過程の詳細
\item 基礎理論に関連するため，自分が勉強した理論・実験法
\item 実験結果の生データ表
\item 実験装置の仕様，操作方法の詳細（マニュアル），実験時の注意点
\item その他，後輩が研究をうまく引き継ぐためのコツやアドバイス
\end{itemize}

などのように，各自が研究を行う中で，本文に記載するには学術的にふさわしくない，同じ内容の繰り返しで量が多く，読者の立場から要点のみの本文への記載がふさわしい，実験時の工夫や苦労など，本文に記述するにはふさわしくないが，後輩のために記録を残したい事柄，を記載すること．付録は，行為目毎に，章を作ること．\LaTeX での原稿作成は，本文と同様に行える．
