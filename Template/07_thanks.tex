\chapter*{謝辞}

謝辞の書き方は，極めて重要．卒論・修論の中で，\textbf{注目度が極めて高い部分}である．研究室外の方に世話にならずに卒研，修士研究を行うことは不可能であるから，研究室外の方に謝辞を書かないで済ませられる学生は，存在しない．謝辞については，以下の文面を必要に応じて書き換えること．

\begin{itemize}
\item 謝辞対象の序列は，①指導教員＞②大学外の研究者＞③他学科の教員＞④所属学科の教員＞④大学職員＞⑤近親者の順が基本である．
\item なお，業者など，主として\textbf{金銭のやりとりを介する場合}には「対価を払った上でのサービスの提供」であるため，\textbf{謝辞を書く必要はない}．
\item 謝辞は，卒業論文（修士論文）が，あくまでも大学外部に対する公開文書であることを意識して書くこと．身内である所属研究室学生に対しては，\textbf{たとえそれが上級生であっても謝辞に名前を挙げることはしない．}同じ研究室の学生の研究に協力することは，「当然」のことである．同じ研究室の同級生や先輩に謝辞を書いている論文を散見するが，対外的な文書である卒論・修論にはふさわしくない，見苦しくさえある．
\end{itemize}
\vspace{5mm}
以下，謝辞例文

\begin{description}
\item[謝辞冒頭] 「終わりに臨み，本研究を進めるにあたり，終始懇切なるご指導とご教示を賜りました指導教員 東海大学工学部航空宇宙学科航空宇宙学専攻 ○○○○　教授（現在，所属している研究室の教員名，2013年度のみ），および水書 稔治 教授（外部卒研の場合は，受け入れ先の責任者名，それに続き，水書の名前）に衷心より感謝の意を表します．」
\item[学外組織研究者に対して] 「本研究の数値解析に有益なご助言とご討論を賜りました　伊藤忠テクノソリューションズ株式会社 阿部 淳 博士（\textbf{組織名，役職名，氏名については，正確に，省略なく書くこと}）に深く感謝の意を表します．」
\item[学内他研究室]に対して 「数値解析では，東海大学工学部航空宇宙学科航空宇宙学専攻 平岡研究室に一部ご協力いただきました．ここに，感謝の意を表します．」
\item[学内他研究室に対して] 「実験では，東海大学工学部航空宇宙学科航空宇宙学専攻　XXXX君　（ただし他研究室であること，所属研究室学生が実験を手伝うのはあたりまえだから謝辞は書かない）にご助力をいただきました．ここに感謝の意を表します．」
\item[学内職員に対して] 「東海大学工学部航空宇宙学科航空宇宙学専攻の職員の方々に様々なご協力をいただき，ここに付記し，感謝の意を表します．」
\item[家族に対して] 「最後に，今まで学生生活のすべての面で支えてくれた家族に心から感謝の意を表します．」
\end{description}
